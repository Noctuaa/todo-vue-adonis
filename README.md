# TodoList Fullstack - Vue 3 + AdonisJS

## 📋 Description
Application complète de gestion de tâches avec frontend Vue 3 et API AdonisJS.

## 🛠️ Technologies
- **Frontend:** Vue 3, Pinia, Vite
- **Backend:** AdonisJS 6, MySQL/PostgreSQL

## 🚀 Installation
### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
